ASP v1.2
Программа создана для фонового воспроизведения информационных оповещений.

Возможности программы: составление планировки для вывода конкретного оповещения в конкретное времяж; выбор и запоминание аудио-устройства для вывода; работа с отдельным I/O позволяет программе работать из трея без влияния на аудио-устройство выбраное в ОС.
